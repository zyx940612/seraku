画面幅の仕様

固定幅：1260px
サイドナビゲーション：360px


「images」フォルダの画像を使用して、「pc_top.png」と同様のレイアウトになるように

WEBページのコーディングを行ってください。



※コーダーの業務にはpsdファイルからの画像切り出し（スライス）が含まれますが、

　今回はコーダーのPCにPhotoshopがインストールされていないケースを想定して

　スライスした画像を用意しています。



※メイン画像のスライダー、トピックス部分のタブ切り替えは

　JavaScriptで実装してください（jQuery+プラグインでの対応可）。
